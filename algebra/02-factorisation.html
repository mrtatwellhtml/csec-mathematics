Algebra 2: Factorisation

This lesson teaches factorisation from the very beginning. Go slowly. Read each explanation. Practise each type. This is a core exam skill used in many other topics.

Learning objectives

  • Factorise by taking out a common factor
  • Recognise and factorise a difference of two squares
  • Factorise trinomials of the form x² + bx + c
  • Factorise trinomials of the form ax² + bx + c
  • Factorise fully
  • Use factorisation to solve equations

1) Common factor

Idea: take out what appears in every term.

Example: 8x² − 4x

Common factors: 4 and x → take out 4x

Answer: 4x(2x − 1)


Common mistake: forgetting the letter when taking out the factor.

2) Difference of two squares

Pattern: a² − b² = (a − b)(a + b)

Example: 4x² − 25 = (2x − 5)(2x + 5)


This only works with subtraction, not addition.

3) Trinomials: x² + bx + c

Find two numbers that multiply to c and add to b.

Example: x² + 7x + 10 → (x + 5)(x + 2)


If c is positive and b is negative, both numbers are negative.

4) Trinomials: ax² + bx + c

Method: ac method (go step by step).

Example: 2x² + 7x + 3 → (2x + 1)(x + 3)


Take your time. Most errors here come from rushing.

5) Factorising fully

Always check for a common factor first.

Example: 6x² − 24 = 6(x² − 4) = 6(x − 2)(x + 2)


6) Solving using factorisation

If ab = 0, then a = 0 or b = 0.

Example: x² − 9 = 0 → (x − 3)(x + 3) = 0 → x = 3 or −3


Write both solutions. Missing one loses marks.

Exam reminder

Factorisation earns method marks. Work clearly, factorise fully, and check your signs.
